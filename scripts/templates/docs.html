Overview

IMMUNOS is a Universal Artificial Immune System that operates as a negative selection model to large language models like Claude. It provides scientific validation, hallucination detection, and multi-modal analysis for rigorous research pursuits.

Primary Function
Negative Selection to Claude
Purpose

Terminal Usage

Quick Start

# Interactive Chat

python3 scripts/immunos_chat.py

Start interactive chat with automatic model routing

# Code Review (FREE via Ollama)

python3 scripts/immunos_assist.py review myfile.py

Review code for bugs, style, security issues using qwen2.5-coder

# Code Generation (FREE via Ollama)

python3 scripts/immunos_assist.py code "sort a list in Python"

Generate code snippets using local models

# Reasoning (FREE via DeepSeek)

python3 scripts/immunos_assist.py reason "explain quantum entanglement"

Deep reasoning tasks using deepseek-r1:14b

# Check Handoff Status

python3 scripts/immunos_handoff.py check

Monitor token usage and handoff requirements
