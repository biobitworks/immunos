# Making Agar Plates

**Category**: Core Protocol
**Version**: 1.0
**Last Updated**: 2025-12-10
**Applicable Labs**: All
**Source**: Community lab standard operating procedures

## Safety Notes

⚠️ **Important**:
- Pour plates before temperature gets below 60°C to prevent clumping
- Add antibiotics, YNB, or Biotin between 60-65°C
- Fill autoclave bottle only ½ full to prevent boil-over
- Agar boil-over can damage the autoclave

## General Guidelines

### Temperature Control
- **Too Hot** (>65°C): Destroys antibiotics
- **Optimal** (60-65°C): Safe to add antibiotics
- **Too Cold** (<60°C): Agar starts to gel/clump

### Antibiotics
- Store in freezer
- Use 1000x stock concentrations
- Add only when media is 60-65°C (touchable but still hot)

## LB Agar Plates

**Purpose**: Growing E. coli

### Ingredients
- LB Agar powder (from media cabinet)
- Follow instructions on container

### Procedure
1. Prepare LB Agar according to container instructions
2. Autoclave
3. Cool to 60-65°C (touchable temperature)
4. Add antibiotic:
   - **Kanamycin** (1000x stock), OR
   - **Ampicillin** (1000x stock)
5. Pour plates (see Pouring Plates section)

## YPD (Yeast Peptone Dextrose) Agar Plates

**Purpose**: Growing yeast

### Ingredients (for 1L)
- 10g Yeast Extract
- 20g Peptone
- 20g Glucose (or Dextrose - same thing)
- 20g Bacteriological Agar
- 1L dH₂O

### Procedure
1. Combine all ingredients in bottle
2. **Add magnetic stirrer** if antibiotics will be added later
   - This allows mixing without creating bubbles from swirling
3. Fill with 1L dH₂O
4. Autoclave
5. Cool to 60-65°C (touchable temperature)
6. If using antibiotics:
   - Add antibiotic from 1000x stock
   - Stir with magnetic stirrer (don't swirl)
7. Pour plates (see Pouring Plates section)

## Minimal Media -HIS Agar Plates

**Purpose**: Selection of transformed yeast

### Background
- Used for GS115 yeast strain
- No amino acids added (GS115 produces all except histidine)
- Insert produces histidine
- Selects for successful transformation

### Ingredients (for 1L)
- 20g Glucose
- 20g Bacteriological Agar
- 900ml dH₂O

### Procedure
1. Combine glucose, agar, and 900ml dH₂O in bottle
2. **Add magnetic stirrer** for later mixing
3. Autoclave
4. Cool to 60°C (any cooler and agar starts to gel)
5. Add:
   - 100ml 10X YNB (Yeast Nitrogen Base)
   - 2ml Biotin 500X
6. Tighten bottle
7. Stir on magnetic stirrer

## Pouring Plates

### Setup
1. Work in the hood
2. Have plates ready

### Procedure
1. Pour ~25ml media per 9cm plate
2. **Keep lid off** until agar solidifies
   - Prevents steam condensation on lid
3. Label bottom of plate (agar side) with:
   - Media type (LB, YPD, Minimal -HIS, etc.)
   - Date
   - Antibiotic (if applicable)

### Storage
- Store plates upside down (agar on top)
- Refrigerate at 4°C
- Use within 2-4 weeks

## YPD Liquid Media

**Purpose**: Liquid culture for yeast growth

### Ingredients (for 1L)
- 10g Yeast Extract
- 20g Peptone
- 20g Glucose (or Dextrose)
- 1L dH₂O

### Procedure
1. Combine all ingredients
2. Fill to 1L with dH₂O
3. Autoclave
4. Cool before use

## Troubleshooting

| Problem | Cause | Solution |
|---------|-------|----------|
| Agar clumping | Poured below 60°C | Pour immediately when media reaches 60°C |
| Bubbles in plates | Swirling instead of stirring | Use magnetic stirrer |
| Antibiotic ineffective | Added when too hot | Wait until 60-65°C before adding |
| Autoclave contamination | Boil-over | Fill bottle max ½ full |
| Plates not setting | Not enough agar | Check agar concentration (typically 1.5-2%) |
